# 🚦 Token System Frontend

A real-time **Token Management System** built using **React + Vite**, designed to efficiently handle queues in environments like hospitals, restaurants, and service centers.

---

## 🌐 Overview

This system allows seamless interaction between:

* 👤 **Customers** – Generate and track tokens
* 🧑‍🔧 **Workers/Admins** – Manage token flow
* 📺 **Display Screen** – Show live queue updates

---

## ⚡ Features

* 🔄 Real-time updates using **Socket.IO**
* 📱 Fully responsive UI
* 🎯 Clean and intuitive dashboards
* ⚡ Fast performance with Vite
* 🟢 Live queue tracking system

---

## 🖥️ Pages

| Route      | Description                            |
| ---------- | -------------------------------------- |
| `/`        | Customer View – Create & track tokens  |
| `/worker`  | Worker Dashboard – Manage token status |
| `/display` | Public Display – Live queue display    |

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🔌 Socket.IO Client

---

## 📦 Installation

```bash
git clone https://github.com/syAnasali/Token_System.git
cd Token_System
npm install
npm run dev
```

---

## ⚙️ Configuration

* Backend URL:

  ```
  http://localhost:3000
  ```

* Update inside:

  * `src/socket.js`
  * `src/api.js`

* Default frontend port:

  ```
  5173
  ```
